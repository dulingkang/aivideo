## 🎞️ 三、详细制作流程（凡人修仙·灵界篇为例）

### **步骤1：准备素材**

✅ 准备以下素材文件夹：

```
/assets
 ├── hanli.png （韩立立绘）
 ├── ziling.png （紫灵立绘）
 ├── lingjie_forest.png （灵界森林背景）
 ├── spirit_mountain.png （灵山背景）
```

可以用 **Leonardo.ai** 或 **Midjourney** 生成，关键词如：

> “cultivator male, calm expression, green robe, spirit realm style, chinese fantasy art, 4k detailed”

---

### **步骤2：生成动画分镜**

使用 **Pika Labs**（[https://pika.art）：](https://pika.art）：)

* 上传场景图和人物图；
* 在文字框输入脚本分镜描述；
* 例子👇：

```
Scene 1: 韩立漂浮在光阵中，空间裂开，光芒吞噬。
Scene 2: 破碎的空间裂缝，灵界山川显现。
Scene 3: 韩立落入山林，环顾四周，目光冷静。
```

Pika会：

* 让人物、光线、雾气动起来；
* 自动生成镜头移动（例如：推近、旋转、俯视）。

导出短片（每镜头5–10秒）。

---

### **步骤3：生成配音**

使用 **ElevenLabs** 或 **ChatTTS**：

* 输入你写好的旁白（比如上面那段云卷仙音解说稿）；
* 选择中文女性声音；
* 调整情感参数（温柔、叙述型）；
* 输出 `.wav` 音频。

> 🎧 提示词：“语气平静、如风中回响、有仙气、叙事感强”。

---

### **步骤4：合成视频**

使用 **Runway ML** 或 **CapCut Pro**：

* 将AI动画片段导入时间线；
* 添加配音；
* 自动生成字幕；
* 调整镜头节奏（建议每句话配1镜头）；
* 背景加淡淡的仙风音乐（可用 Soundraw 自动生成“fantasy ambient”风格）。

---

### **步骤5：导出与包装**

* 封面：可用 **Canva / Fotor / BlueWillow** 做“云卷仙音系列封面”。
  封面元素：韩立背影 + 灵界山河 + 标题「凡人修仙·灵界篇① 飞升之门」
* 导出格式：1080p，2–3分钟长度；
* 上传时加话题标签：
  #凡人修仙传 #AI动画 #云卷仙音 #灵界篇
