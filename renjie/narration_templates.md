# 旁白文字模板

## 开头旁白模板（固定格式）

**模板：**
```
我是云卷仙音，欢迎回来。第一百{X}集《{标题}》。
```

**示例：**
- 第171集：`我是云卷仙音，欢迎回来。第一百七十一集《南陇侯的邀约·收仆赴宴》。`
- 第172集：`我是云卷仙音，欢迎回来。第一百七十二集《{新标题}》。`

**变量：**
- `{X}`：集数（中文数字）
- `{标题}`：本集标题

## 结尾旁白模板（固定格式）

**模板：**
```
下集再见。我是云卷仙音，第一百{X+1}集的故事即将开始。
```

**示例：**
- 第171集：`下集再见。我是云卷仙音，第一百七十二集的故事即将开始。`
- 第172集：`下集再见。我是云卷仙音，第一百七十三集的故事即将开始。`

**变量：**
- `{X+1}`：下一集集数（中文数字）

## 使用建议

1. **统一格式**：所有集数使用相同格式，保持一致性
2. **只替换变量**：只需要替换集数和标题，其他文字固定
3. **自动生成**：可以编写脚本自动根据集数和标题生成旁白

## 可选的变体格式

如果希望更简洁：

**简洁开头：**
```
欢迎回来。第一百{X}集《{标题}》。
```

**简洁结尾：**
```
下集再见。第一百{X+1}集的故事即将开始。
```
