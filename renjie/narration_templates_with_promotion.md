# 旁白文字模板（包含推广）

## 开头旁白模板（固定格式，不含推广）

**模板：**
```
我是云卷仙音，欢迎回来。第{X}集《{标题}》。
```

**示例：**
- 第171集：`我是云卷仙音，欢迎回来。第一百七十一集《南陇侯的邀约·收仆赴宴》。`
- 第172集：`我是云卷仙音，欢迎回来。第一百七十二集《{新标题}》。`

## 结尾旁白模板（固定格式，包含推广，顺序优化后）

### 方案1：简洁版（推荐）✅
**模板：**
```
我是云卷仙音，第{X+1}集的故事即将开始。喜欢的话请点赞支持一下！下集再见。
```

**示例：**
- 第171集：`我是云卷仙音，第一百七十二集的故事即将开始。喜欢的话请点赞支持一下！下集再见。`

**逻辑顺序：**
1. 介绍下一集（预告）
2. 请求支持（推广）
3. 说再见（结束）

### 方案2：完整版
**模板：**
```
我是云卷仙音，第{X+1}集的故事即将开始。如果喜欢的话，请多多支持、点赞、订阅！下集再见。
```

**示例：**
- 第171集：`我是云卷仙音，第一百七十二集的故事即将开始。如果喜欢的话，请多多支持、点赞、订阅！下集再见。`

### 方案3：超简洁版
**模板：**
```
我是云卷仙音，第{X+1}集的故事即将开始。喜欢的话请多多支持！下集再见。
```

**示例：**
- 第171集：`我是云卷仙音，第一百七十二集的故事即将开始。喜欢的话请多多支持！下集再见。`

## 推广文案变体

如果希望多样化，可以使用以下变体（但建议固定使用一种保持一致性）：

1. `如果喜欢的话，请多多支持、点赞、订阅！`
2. `喜欢的话请点赞、订阅，支持一下！`
3. `喜欢的话请点赞支持一下！`
4. `喜欢的话请多多支持！`
5. `如果觉得不错，请点赞订阅支持！`

## 使用建议

1. **固定格式**：所有集数使用相同格式，保持一致性
2. **推广位置**：建议放在结尾，不影响开头节奏
3. **推广长度**：根据时长选择合适版本（完整版/简洁版）
4. **统一使用**：建议选择一种推广文案，所有集数统一使用

## 时长考虑

- **开头**：3-5秒（不含推广）
- **结尾**：5-7秒（含推广，建议使用简洁版或超简洁版）
