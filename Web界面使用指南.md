## 🚀 快速开始（Gradio版）

### 1. 确保API服务运行

```bash
# 启动API服务（如果未运行）
python gen_video/api/mvp_main.py
```

### 2. 启动Web界面

```bash
./启动Web界面.sh
```

### 3. 访问界面

打开浏览器：http://localhost:7860

### 4. 使用界面

1. **输入API Key**：默认 `test-key-123`
2. **查询配额**：点击"查询配额"按钮
3. **生成图像**：
   - 切换到"图像生成"标签页
   - 输入提示词
   - 调整参数（可选）
   - 点击"生成图像"
   - 等待30-60秒
   - 查看结果

## 📋 使用技巧

### 1. 提示词编写

**好的提示词**：
- 详细描述：包括风格、颜色、构图、情绪等
- 具体场景：描述具体的场景和元素
- 风格指定：如"动漫风格"、"写实风格"等

**示例**：
```
一个美丽的风景，山峦起伏，云雾缭绕，阳光透过云层，动漫风格，色彩鲜艳，细节丰富
```

### 2. 参数调整

- **推理步数**：40-50步通常效果最好
- **引导尺度**：7-8适合大多数场景
- **分辨率**：1024x1024是平衡质量和速度的最佳选择

### 3. 负面提示词

描述不想要的内容：
- 模糊、低质量
- 变形、扭曲
- 不想要的元素

## ⚠️ 注意事项

### 1. API服务必须运行

Web界面需要API服务支持，确保API服务在 `http://localhost:8000` 运行。

### 2. 生成时间

- 图像生成：通常需要30-60秒
- 请耐心等待，不要重复点击

### 3. 配额限制

- 免费版：10图/天
- 配额每天自动重置
- 配额用完需要等待或升级

### 4. 浏览器兼容

- 推荐使用Chrome、Firefox、Edge等现代浏览器
- 不支持IE浏览器

## 🐛 常见问题

### Q1: 界面无法访问

**A**: 检查：
1. API服务是否运行（http://localhost:8000）
2. Web界面是否启动（http://localhost:7860 或 3000）
3. 防火墙是否阻止端口

### Q2: 生成失败

**A**: 检查：
1. API Key是否正确
2. 提示词是否为空
3. 配额是否用完
4. 查看控制台错误信息

### Q3: 图像不显示

**A**: 检查：
1. 生成是否成功（查看状态信息）
2. 网络连接是否正常
3. 浏览器控制台是否有错误
