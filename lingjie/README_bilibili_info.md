# B站发布信息生成说明

本目录包含自动生成的B站发布信息，用于《凡人修仙传·灵界篇》系列视频的发布。

## 文件说明

### 生成的文件

1. **`bilibili_release_info.md`** - Markdown格式的发布信息
   - 包含每一集的标题、描述、标签等信息
   - 方便阅读和复制粘贴
   - 适合手动发布时参考使用

2. **`bilibili_release_info.json`** - JSON格式的发布信息
   - 机器可读的格式
   - 方便程序化处理或批量发布
   - 每集包含完整的发布信息

3. **`generate_bilibili_info.py`** - 生成脚本
   - 用于自动生成发布信息
   - 可以从scenes目录下的JSON文件提取信息
   - 支持自定义标题翻译和描述格式

## 使用方法

### 查看发布信息

直接打开 `bilibili_release_info.md` 文件，找到对应集数，复制标题和描述即可。

### 重新生成信息

如果需要更新发布信息（例如修改了scenes目录下的JSON文件），运行：

```bash
cd /vepfs-dev/shawn/vid/fanren/lingjie
python3 generate_bilibili_info.py
```

### 使用JSON格式

如果需要进行程序化处理，可以使用JSON格式：

```python
import json

with open('bilibili_release_info.json', 'r', encoding='utf-8') as f:
    episodes = json.load(f)

# 获取第1集信息
ep1 = next(e for e in episodes if e['episode'] == 1)
print(f"标题：{ep1['bilibili_title']}")
print(f"描述：{ep1['description']}")
print(f"标签：{', '.join(ep1['tags'])}")
```

## 发布信息结构

每一集的发布信息包含：

- **episode**: 集数
- **title_cn**: 中文标题
- **title_en**: 英文标题（原始）
- **bilibili_title**: B站格式标题（包含【凡人修仙传·灵界篇】前缀）
- **description**: 完整的视频描述
  - 剧情要点
  - 系列信息
  - 主要角色
  - 原著信息
  - 互动提示
  - 标签（Hashtag格式）
- **tags**: 标签列表（用于B站标签功能）

## 标题来源

标题主要从以下来源获取，按优先级排序：

1. **EPISODE_TITLE_MAP** - 脚本中的集数到标题映射（最准确）
2. **TITLE_TRANSLATION** - 英文标题到中文标题的翻译映射
3. **章节对应记录.md** - 参考文档中的标题信息
4. 自动处理 - 如果以上都没有，会尝试自动处理英文标题

## 注意事项

1. **JSON文件格式错误**：如果某些JSON文件有格式错误（如13.json, 15.json, 44.json），脚本会跳过这些文件并显示警告。需要先修复这些文件的JSON格式。

2. **标题翻译**：如果发现标题翻译不准确，可以在 `generate_bilibili_info.py` 中的 `EPISODE_TITLE_MAP` 或 `TITLE_TRANSLATION` 中添加或修改映射。

3. **描述内容**：描述中的剧情要点是从JSON文件中提取的关键旁白。如果需要调整描述内容，可以修改 `generate_description()` 函数。

4. **标签**：标签列表可以根据需要进行调整。B站标签通常限制在10个以内。
